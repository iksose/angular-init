angular-init
============

* Koa
* gulp
* browserify
* stylus
* angular

#### Install
  ```$ clone && npm install && bower install && npm install gulp -g```

#### Run
  ```$ gulp ```

#### Test
  ```$ gulp karma```

This will test in Chrome and IE8 (WinXP) via Virtualbox. You can install IE8 via ```node_modules/.bin/iectrl install 8,WinXP```
